# 🌾 HarvestIQ
 
 AI-powered agricultural decision support system for fertilizer recommendation, crop insurance prediction, price forecasting, and yield prediction. 🌱🚜📊

 ---

## 📌 Project Overview
Smart Farming AI is a Machine Learning based agriculture dashboard that helps farmers with:

- 🌾 Crop Recommendation
- 📈 Yield Prediction
- 💰 Price Prediction
- 🧪 Fertilizer Recommendation
- 📜 Insurance Prediction

Built using Streamlit for an interactive and user-friendly interface.

---

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- XGBoost

---

## Objective

To provide a unified AI-driven platform that empowers farmers with actionable insights for smarter agricultural planning and management.
